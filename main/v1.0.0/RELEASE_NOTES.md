This is the first release of Snakension, a browser extension that brings the classic Snake game to your browser.

## ‚ú® Features

- **Snake Gameplay**: A smooth and classic Snake game experience.
- **Offline Auto-Notify**: The extension automatically notifies when your browser detects no internet connection‚Äîno more boring "no internet" pages!
- **High Score Tracking**: Your best score is automatically saved so you can try to beat it next time.
- **Dual Controls**: Play using **Arrow Keys** or **WASD**.
- **Pause/Resume/Reset**: Full control over your game session with on-screen buttons and Spacebar shortcuts.
- **Connection Status**: Real-time indicator showing if you are Online or Offline.
