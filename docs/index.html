Get Started

Installing Snakension is quick and easy. Works on Chrome, Edge, Brave, or any chromium based browser.

1

Download the Package

Download the Zip File from the latest releases and unzip it on your computer.

2

Open Extensions Page

Navigate to chrome://extensions/ (or edge://extensions/ if you're on Edge), or open extensions manager manually.

3

Enable Developer Mode

Toggle the Developer mode switch in the top-right corner of the extensions page.

4

Load Unpacked

Click the Load unpacked button and select the snakension folder you unzipped.
